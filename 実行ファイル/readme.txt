学生時代に初オリジナル作成したゲームとなります。
そのため、説明が不足していたのでこちらで追記いたします。

■■■■■■■■■■
タイトル画面にてEnterを押し、十字キーをもちいて選択移動、Enterで確定します。

遊び方ページではXキーで次のページに進み、
Zキーで前のページに戻ります。

ゲーム画面では、
・左上の丸がライフ
・右上のバーがMPとなり、属性の玉を打つと消費されます。
■■■■■■■■■■

都度、セキュリティ通知が発生するようです。
VisualStudioにて開発したプロジェクトファイルも添付いたします。コンパイルしていただけますとセキュリティ通知は発生しないと思われます。
